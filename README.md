# userscripts

A bunch of userscripts for various sites, either made for my use or made under a request.

Requires [Tampermonkey](https://www.tampermonkey.net/) (or a compatible userscript manager).

---

## rule34.xxx

| Script | Description | Install |
|---|---|---|
| Kivl's Improvements | A bunch of improvements for the Rule34.xxx website created by Kivl | [Install](https://github.com/mja00/userscripts/raw/main/rule34/kivls-improvements.user.js) |
| Forum Improvements | Various forum improvements | [Install](https://github.com/mja00/userscripts/raw/main/rule34/forum-improvements.user.js) |
| Better Forum Timestamps | Adds relative timestamps to forum posts and converts them to local time | [Install](https://github.com/mja00/userscripts/raw/main/rule34/better-forum-timestamps.user.js) |
| Navbar Improvements | Adds and improves on the navbar | [Install](https://github.com/mja00/userscripts/raw/main/rule34/navbar-improvements.user.js) |
| Image Size | Shows image size on the statistics pane | [Install](https://github.com/mja00/userscripts/raw/main/rule34/more-image-info.user.js) |
| New Mail Notification | Sends a desktop notification when you receive new mail | [Install](https://github.com/mja00/userscripts/raw/main/rule34/mail-notifications.user.js) |
